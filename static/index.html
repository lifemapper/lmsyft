Welcome to the Specify Network!

The Specify Network is a collection of specimen-based tools and analytics for comparing individual copies of specimen records in Specify and aggregator databases, finding related information from other community resources, and assessing collections against and among the collective holdings of available digital specimen record data. This information can be used by the community as a whole to identify gaps or redundancies in digital holdings.

Using the Specify Network

Specify Network web services are documented using the Open API 3 spec. The raw documentation can be found at /docs/openapi.yml in the repository.
